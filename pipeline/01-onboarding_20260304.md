# スキル: 初回オンボーディング

## トリガー: `/start`

## 処理フロー（最大5ステップ）

### Step 1: ヒアリング（1ターン目）
以下を質問（一度にまとめて聞く）:
- 現在の役職・業界
- AI活用の経験レベル（初心者/中級/上級）
- 特に興味のある分野（技術/戦略/組織/倫理 から複数選択）
- 1日に使える学習時間（15分/30分/60分）
- 学習の目標（自由記述）

### Step 2: プロファイル生成（2ターン目、ユーザー回答後）
`data/profile.json` を作成:
```json
{
  "created": "ISO日付",
  "name": "ユーザー名",
  "role": "役職",
  "industry": "業界",
  "ai_level": "beginner|intermediate|advanced",
  "interests": ["技術", "戦略"],
  "daily_minutes": 30,
  "goal": "自由記述",
  "streak": { "current": 0, "best": 0, "last_date": null }
}
```

### Step 3: 推奨学習パス提示
`skills/02-topics.md` のトピック一覧から、ユーザーのレベルと興味に基づき最初の5トピックを推奨。

### Step 4: progress.json 初期化
```json
{
  "topics": {},
  "spaced_queue": [],
  "metacognition": { "total_checks": 0, "overconfident": 0, "calibration_trend": [] }
}
```

### 応答テンプレート
```
🔥 ARCHITECT FORGE へようこそ

プロファイルを作成しました。あなたの学習パスを提案します:

📋 推奨スタートトピック:
1. [T0XX] トピック名 — 理由の一行説明
2. ...（最大5つ）

次のステップ:
- `/learn T0XX` で教材を生成して学習開始
- `/topics` で全トピック一覧を確認
- `/session` でガイド付き学習セッション
```
