## `/review` — デイリー復習セッション

### 処理フロー（最大5ステップ）

1. `data/progress.json` を読む
2. 間隔反復キュー（`spaced_queue`）から今日の復習対象を最大5件抽出
3. インターリービング: 異なる分野から交互に出題
4. 各問題を1つずつ提示（ユーザー回答→フィードバックの対話形式）
5. 全問終了後、スコアと間隔更新を `progress.json` に保存

### 間隔反復アルゴリズム（SM-2簡易版）

```
正答時:
  repetitions += 1
  interval = 前回interval × ease_factor
  ease_factor = max(1.3, ease_factor + 0.1)

誤答時:
  repetitions = 0
  interval = 1（翌日に再出題）
  ease_factor = max(1.3, ease_factor - 0.2)

スケジュール目安: 1日→3日→7日→14日→30日→90日
```

### 出題形式（ランダム選択）

1. **フラッシュリコール**: 「[トピック名]の核心を3つ挙げてください」
2. **精緻化質問**: 「[概念X]はなぜ機能するのですか？メカニズムを説明してください」
3. **文脈転換**: 「[概念X]を[異なる業界]に適用するとどうなりますか？」
4. **比較分析**: 「[トピックA]と[トピックB]の共通点と相違点は？」
5. **批判的評価**: 「[理論X]の最大の限界は何ですか？」

### メタ認知チェック（各問題の前後）

```
[問題提示前]
「この問題にどの程度自信がありますか？（1-5）」

[回答・フィードバック後]
自信度と実際スコアの差を記録
calibration_gap = confidence - actual_score
```

## `/quiz [ID]` — 特定トピックのテスト

### 処理フロー

1. `data/materials/[ID].md` を読む（教材がなければ生成を促す）
2. SYNTHESIS CHALLENGE の3問 + 追加2問を生成
3. Bloomのタキソノミーの複数レベルから出題:
   - Remember: 定義・事実の想起
   - Understand: メカニズムの説明
   - Apply: 新しい文脈への適用
   - Analyze: 要素の分解・比較
   - Evaluate: 判断・批判
4. 採点（0-5点/問）→ マスタリーレベル更新

### マスタリーレベル更新基準

| レベル | 名前 | 条件 |
|--------|------|------|
| 0 | UNAWARE | 未接触 |
| 1 | EXPOSED | 教材を読んだ |
| 2 | FAMILIAR | 復習2回 + 正答率50%以上 |
| 3 | COMPETENT | 復習4回 + 正答率70%以上 + 適用問題正答 |
| 4 | PROFICIENT | 復習6回 + 正答率85%以上 + 分析問題正答 |
| 5 | EXPERT | 復習8回 + 正答率90%以上 + 評価問題正答 + プロテジェセッション完了 |
| 6 | ARCHITECT | 復習10回 + 正答率95%以上 + 独自フレームワーク提案 + 実務適用報告 |

## `/teach [ID]` — プロテジェ効果セッション

### 処理フロー

1. AIが「このトピックを何も知らない新人」を演じる
2. ユーザーが教える → AIが初学者として質問を投げる
3. 答えられない質問 = 理解のギャップ → 復習対象に追加
4. セッション後にギャップレポートを表示

### AIの質問パターン
- 「それは具体的にどういうことですか？」
- 「なぜそれが重要なのですか？」
- 「例を1つ教えてください」
- 「〇〇の場合でも同じことが言えますか？」
- 「他の方法ではダメなのですか？」

最大5往復で区切る。

## `/socratic [ID]` — ソクラテス式対話

### 処理フロー

1. トピックに関連するケーススタディを1つ提示
2. AIは答えを与えず、問いを重ねる
3. 思考の深さに応じて質問レベルを動的調整

### 質問の段階
```
Level 1（事実確認）: 「何が起きたのですか？」
Level 2（理解）: 「なぜそうなったと思いますか？」
Level 3（分析）: 「別の要因は考えられますか？」
Level 4（評価）: 「あなたがCDOならどう判断しますか？」
Level 5（創造）: 「この事例から独自のフレームワークを作れますか？」
```

最大8往復で区切り、最後にAIが思考の深さをフィードバック。

### 更新タイミング
- `/review` 完了時: spaced_queue の interval/ease_factor/next_review を更新
- `/quiz` 完了時: topics のマスタリーレベルを更新
- `/teach` 完了時: ギャップを spaced_queue に追加、teach_completed フラグ
- `/socratic` 完了時: 最高到達レベルを記録
